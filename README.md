## 使用方法：

一：

1. 克隆仓库源代码

   2.删除.git文件夹，如果有自定义域名请修改根目录source/CNAME中的内容

   3.打开根目录_config.yml修改仓库地址

   4.根据自己需要配置一下就可以使用啦
